# Smart Parking Management System

## Overview
The Smart Parking Management System uses AI and IoT to detect vacant parking spaces, guide drivers to available slots, and reduce traffic congestion.

## Features
- Real-time parking detection
- Slot availability
- AI-based parking prediction
- User dashboard
- Smart notifications

## Technologies Used
- Python
- Streamlit
- OpenCV
- Machine Learning

## Future Scope
- Mobile booking
- QR-based parking
- Smart city integration
- Digital payments
